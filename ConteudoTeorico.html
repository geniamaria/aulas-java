1. Objectos

Java é uma linguagem de programação orientada a objetos. Quando você trabalha em Java, você usa principalmente objetos para realizar o trabalho. Você cria objetos, modifica-os, alterar suas variáveis, chamar seus métodos e combinar-los com outros objetos.

1.1. Criar Objectos

Ao escrever um programa Java, você define um conjunto de classes. Uma classe é um modelo usado para criar um ou mais objectos. Esses objectos, que também são chamados de instâncias, são elementos autocontidos de um programa com recursos e dados relacionados. Na maioria das vezes, você usa uma classe apenas para criar instâncias (objectos) e depois trabalhar com essas instâncias.

Por exemplo, uma string (uma série de caracteres entre aspas duplas) pode ser usada para criar uma nova instância da classe String com o valor dessa string. A classe String é incomum nesse aspecto. Embora seja uma classe, pode ser atribuído a ela um valor com um literal como se fosse um tipo de dados primitivo.


Para criar um novo objecto, você usa o operador new com o nome da classe que deve ser usado como seu modelo. O nome da classe é seguido de parênteses, como nos seguinte dois exemplos:

String nome = new String("Jay Ubisse");
URL endereco = new URL("https://learn-jay-ubisse.vercel.app/"); Carro novoCarro = new Carro();

Os parênteses são importantes e não podem ser omitidos. Eles podem estar vazios, no entanto, em nesse caso, o objeto mais simples da classe especificada é criado. Os parênteses também pode conter argumentos que determinam os valores de variáveis ​​de instância ou outras variáveis ​​iniciais qualidades desse objeto.

O número e tipo de argumentos a serem incluídos entre parênteses são definidos pela classe em si, usando um método especial chamado construtor (que será apresentado posteriormente nesta lição). Se você tentar criar uma nova instância de uma classe com o número errado de argumentos ou um tipo errado de argumentos, ou se você não fornecer argumentos e precisar deles, ocorrerá um erro quando o programa é compilado.

O primeiro projeto da lição demonstra como criar diferentes tipos de objetos com diferentes números e tipos de argumentos. A classe StringTokenizer no O pacote java.util divide uma string em uma série de strings mais curtas chamadas tokens.
