1. Classes e Objectos

1.1. Modificadores de acesso

  • public => Pode ser acedido por qualquer outro programa Java
  • protected => pode ser acedido apenas por programas java que estao no mesmo pacote
  • private => Nao pode ser acedido por programas externos

1.2. Variaveis e metedos estaticos

Todo o metodo estatico deve chamar e/ou usar variaveis ou metodos tambem estaticos.

3. Classes - Sao modelos instanciaveis, i.e, modelos a partir dos quais podemos criar copias e usar os seus atributos e/ou metodos. Classe e' um modelo para a criacao de um objecto. 3.1. Atributos - sao propriedades que uma classe tem. 3.2. Construtor - e' um metodo que inicializa os atributos da classe, ou seja, da as informacoes inciais dos atributos da classe. NB: Os construtores dever ser publicos e devem ter o mesmo nome que a classe. 3.3. Getters e Setters - sao usados para obter e modificar valores dos atributos respectivamente. 4. Objectos - objectos sao instancias de uma classe. (Um objecto e' uma copia de uma classe) 4.1. Sintaxe da criacao de um objecto NomeDaClasse nomeDoObjecto = new NomeDaClasse(argumentos);

3. Heranca

Heranca e' a capacidade de uma classe herdar atributos e metodos de uma classe. A classe que herda os atributos e metodos e' chamada de subclasse e outra classe e' chamada superclasse.
Para herdar os atributos e metodos de uma classe, usamos a palavra-chave "extends".
